# Pitch Rules

## Service-First Rule

If recent history contains unresolved issues, product rejection, email bounce, category mismatch, or Buy Lead consumption concerns, start with issue resolution before any upsell.

## Evidence Rule

Every pitch must include at least one piece of seller-specific evidence:

- recent call/meeting note
- performance metric
- pain point
- package/service indicator
- product/category signal

## Safety Rule

Do not reveal unmasked contact details. Use masked values exactly as provided.

## Hinglish Example

Sir, aapke account me activity strong hai, lekin recent history me Buy Lead consumption aur export leads ka issue repeat hua hai. Main pehle woh verify karwa deta hoon, phir category preference aur product specifications optimize karte hain taaki relevant enquiries improve ho sakein.
